# Introduction

Thermalith is a free, local-first desktop app for designing and printing labels on NIIMBOT
thermal printers. You lay a label out on screen exactly as it will print, then send it straight
to the printer over USB — no account, no cloud, and no telemetry. Everything stays on your machine.

Thermalith runs on Windows, macOS, and Linux.

## What you can do

- Set up a label to match your roll (size, shape, resolution).
- Add and arrange text, barcodes, QR codes, serial numbers, dates and times, shapes, lines,
  images, and tables.
- See an accurate, print-true preview as you work — what you see is what burns onto the label.
- Connect a NIIMBOT printer, check its status, set print density, and print.

## A note on printers

Thermalith talks to NIIMBOT printers directly. The **B1** is the fully verified reference model;
other models are supported through the built-in printer catalogue, with B1 remaining the model
that has been confirmed end-to-end on real hardware. Thermalith is an independent project and is
not affiliated with NIIMBOT.

## Free and donation-supported

Thermalith is open source under the GPL-3.0 licence and is free for everyone. If it saves you
time, there is an optional **Donate** option in the About window — entirely voluntary.

The rest of this manual walks through the window, building a label, each element type, and printing.
