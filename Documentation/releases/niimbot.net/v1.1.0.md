`Niimbot.Net` is the standalone C# protocol + transport library for NIIMBOT thermal printers (GPL-3.0),
independent of any UI or label engine. Thermalith is its first consumer.

## Added

- **D110M-v4 print task** for the D11_H (shared by the B1 Pro / B21 Pro / D110-M v4): a 9-byte
  `PrintStart` and a 13-byte `SetPageSize` that carries the copy count, with one-way `PrintStatus` /
  heartbeat framing. Selected automatically by model id. Hardware-verified on a real D11_H.
- `PrintStart9b` and `SetPageSize13b` packet builders.

## Fixed

- D11_H (id 528) catalogue geometry corrected to **300 dpi / 142 px** printhead (was 229 / 108, the
  229 having been back-computed from the wrong head width), so rasters size correctly.

## Notes

- The public API is otherwise unchanged from 1.0.0, and the B1 print path is byte-for-byte identical —
  this is an additive, backward-compatible release.
